`npm install<ten_thu_vien>`

`npm unstall<ten_thu_vien>`

# sau khi cài đặt xong thư viện 
- liệt kê các thư viện được cài đặt trong file package.json
- Tạo ra folder node_module: chứa các thư viện được cài đặt trong dự án 
- Tạo ra file package-lock.json 
# so sánh file package.json và package-lock.json
- package.json: liệt kê khoảng verstion được phép cài đặt trong dự án (liệt kê trong khoảng nào đó )
- package-lock.json: liệt kê chính xác verstion mà chúng ta cài đặt trong dự án 
# .gitignore 
- loại bỏ những folder hay file ko muốn commit lên git</br>
- loại bỏ node_modules vì quá nặng.</br>
